El proyecto es una página web para el Centro Comercial Arenales para poder ver productos, tiendas y poder realizar pedidos con fecha de pedido y entrega

Se utilizo:

- ASP Net Core 2.1
- Entity Framework
- Angular 6
- Bootstrap 4

- Visual Code (frontend)
- Visual Studio (backend)

Pasos para levantar el proyecto:

1. Levantar el proyecto de la carpeta "Backend" (ArenalesStore.sln")
2. Ejecutar el proyecto
3. Abrir con VS Code la carpeta "...\Frontend\ArenalesStore"
4. Ejecutar "npm install && ng serve" dentro de la carpeta
5. Abrir en navegador "http://localhost:4200/"
